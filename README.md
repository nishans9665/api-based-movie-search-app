This is a responsive movie search application built using JavaScript, Bootstrap, and the OMDb API.  
Users can search movies by name, browse keyword shortcuts, discover random movies, and view top-rated movie lists.

## Features

### Movie Search
- Search movies by typing a name and pressing Enter.
- Displays results instantly inside the Search Results section.
- Each movie item shows:
  - Poster
  - Title
  - Year
- Clicking a movie opens the detailed movie page.

### Keyword Search
Shortcut buttons for quick searches:
- Batman
- Love
- War
- Action
- Comedy

### Discover Random Movie
- Selects a random keyword and loads movies from that category.

### Top Rated Movies
- Automatically loads a collection of the highest-rated classic movies.
- Each card includes:
  - Poster
  - Title
  - IMDb rating

### Movie Details Page
- Displays:
  - Poster
  - Title
  - Year
  - Genre
  - Director
  - Actors
  - Plot
  - IMDb Rating

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5.3
- JavaScript
- OMDb API

## API
This project uses the OMDb API.
